# 书城

## 技术栈

gulp + handlebars + jquery + require + ES6/7 + text + sass + flex

## 项目运行

#### 注意：由于涉及大量的 ES6/7 等新属性，node 需要 6.0 以上版本 

```
git clone https://github.com/bailicangdu/vue2-elm.git  

cd vue2-elm

npm install  或 yarn(推荐)

npm run dev

```
